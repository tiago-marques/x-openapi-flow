# x-openapi-flow Examples

This directory contains one project per supported integration:

- `swagger-ui/`: local Swagger UI app that loads `swagger.flow.json`.
- `redoc/`: static Redoc output generated by `x-openapi-flow generate-redoc`.
- `postman/`: Postman collection generated by `x-openapi-flow generate-postman`.
- `insomnia/`: Insomnia export generated by `x-openapi-flow generate-insomnia`.
- `sdk/typescript/`: flow-aware TypeScript SDK generated by `x-openapi-flow generate-sdk`.

## Goal

Each project helps you validate the same `x-openapi-flow` lifecycle:

1. Start from base OpenAPI (`swagger.json`).
2. Apply sidecar (`examples/swagger.x.yaml`) to create `swagger.flow.json`.
3. Validate with strict profile.
4. Generate and test the tool-specific artifact.

## Quick start by project

- `swagger-ui/`: run local docs UI and verify flow fields in rendered operations.
- `redoc/`: generate static docs package and serve it locally.
- `postman/`: generate collection and import in Postman.
- `insomnia/`: generate workspace export and import in Insomnia.
- `sdk/typescript/`: generate TypeScript SDK and run a minimal sample consumer.

See each subfolder README for exact commands.
