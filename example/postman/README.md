# Postman Example

Use this project to create and test a Postman collection from `x-openapi-flow`.

## What this project covers

- Build `swagger.flow.json` from `swagger.json` + sidecar.
- Validate flow-enriched OpenAPI with strict rules.
- Generate `x-openapi-flow.postman_collection.json` for Postman import.

## Key files

- `swagger.json`: base OpenAPI input.
- `examples/swagger.x.yaml`: sidecar with `x-openapi-flow` metadata.
- `swagger.flow.json` (generated): merged OpenAPI + flow data.
- `x-openapi-flow.postman_collection.json` (generated): Postman collection.

## Create flow output

```bash
cd /workspaces/x-flow/example/postman
npm install
npm run diff
npm run apply
npm run validate
```

## Generate Postman artifact

```bash
npm run generate
```

## Test in Postman

1. Import `x-openapi-flow.postman_collection.json` into Postman.
2. Configure a collection/environment variable such as `baseUrl`.
3. Execute requests in the expected order and verify transition behavior.
4. Validate scripts generated by `--with-scripts` as part of your flow checks.

## Equivalent direct CLI command

```bash
cd /workspaces/x-flow
npx x-openapi-flow generate-postman example/postman/swagger.flow.json --output example/postman/x-openapi-flow.postman_collection.json --with-scripts
```
